
 # 전체 종목 가져오는 방법
 '한국거래소' 사이트 이용 

 http://marketdata.krx.co.kr

 시장정보 -> 상장현황 -> 상장회사 검색 으로 들어가서 '시장구분' 에서 '전체' 클릭 후 '조회' 버튼 클릭 한 다음 Excel 또는 csv 로 다운로드 

 다운받은 파일 data.csv, data.xls
 
 개행오류 수정 파일 : KOSDAQ.csv, KOSPI.csv
